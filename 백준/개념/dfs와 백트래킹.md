# 🔎 DFS vs 백트래킹
- DFS: 단순히 "깊이 우선으로 탐색하는 방법"
- 백트래킹: DFS의 한 응용 기법
- 탐색 중간에 조건을 만족하지 않으면 가지치기(pruning) 해서 불필요한 탐색을 줄임
- 예: N-Queen, 순열/조합 문제, 최적화 문제
- 즉, 백트래킹은 DFS를 기반으로 하지만 "조건 검사 + 되돌리기(pop)"를 추가한 것

# 🔎 DFS vs BFS
 - DFS: "모든 경우를 다 봐야 할 때" → 완전탐색, 백트래킹
 - BFS: "최단 거리나 레벨 탐색이 필요할 때" → 거리 계산, 확산 문제
