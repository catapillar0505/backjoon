 => 리스트 안의 리스트/튜플속 숫자쌍을 각각 꺼내어 변수로 담을 수있음

 for문에서도 활용 가능

 - counter.items()
→ Counter(혹은 일반 dict)에서 (키, 값) 쌍들을 묶어서 보여주는 뷰 객체야.
→ 정확히는 dict_items라는 특별한 타입이지, 리스트는 아니야.
- 그런데 파이썬의 sorted() 함수는 리스트만 정렬하는 게 아니라, 반복 가능한(iterable) 모든 것을 정렬할 수 있어.
→ dict_items도 iterable이니까, sorted()에 넣으면 내부적으로 하나씩 꺼내서 정렬해준다.
→ 정렬 결과는 새로운 리스트로 반환돼.
